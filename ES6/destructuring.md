# ⚒️  `디스트럭처링 알아보자`

<br>

* **정의**
* **사용법**
* **사용 이유**

<br>

> 정의

```
배열이나 객체의 속성을 해체하여 
개별 변수로 추출하여 할당하는 기능
```
<br>

> 사용법

<br>

&nbsp;&nbsp;&nbsp;&nbsp;`배열`

```
const [el1, el2, ...rest] = array;
```

<br>

&nbsp;&nbsp;&nbsp;&nbsp;`객체`

```
const { prop1, prop2 } = object;

```

<br>

> 사용이유


```
* 변수 할당 간소화

* 코드 가독성 향상

* 중첩된 구조 해체
```
