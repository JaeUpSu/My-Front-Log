# ⚒️  `Arrow Function 알아보자`

<br>


* **정의**
* **사용이유**
* **기능**

<br>

> 정의


```
npx create-react-app my-app-name --template typescript

npx create-next-app@latest my-app-name --template typescript
```
<br>

> 사용이유

<br>

`TypeScript 컴파일러를 구성하고 TypeScript 프로젝트에 대한 옵션을 지정하는 데 사용`

- tsConfig
    - 컴파일러 옵션
    - 포함 및 제외
    - 추가 옵션
    
<br>


`컴파일러 옵션`

```
{
  "compilerOptions": {
    "target": "es5",
    "module": "commonjs",
    "outDir": "dist",
    "rootDir": "src",
    "strict": true
  }
}
```
